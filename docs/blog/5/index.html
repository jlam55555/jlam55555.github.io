Installed packages on a (fairly) new Debian install

On 4/19/2021, 5:31:25 PM

Return to blog

I recently reinstalled Debian 10 (buster, i.e., the stable version) due to a dependency disaster stemming from an accidental upgrade from bullseye to bullseye/sid that upgraded my CUDA version which caused incompatibility with my 8-yo Nvidia card...

I've been using this new system for a few days already, but I thought it'd be interesting to store all the packages somewhere for me to reflect on. Of course, most things are installed with apt. The only exceptions are MATLAB, the IntelliJ's (CLion, IDEA Community, PyCharm, WebStorm, and GoLand), and Python 3.8 (to install a newer pip version needed for TensorFlow 2).

Listing packages manually installed with apt

grep 'apt install' /var/log/apt/history.log \
	| cut -f 4- -d ' ' - \
	| sed 's/ /\n/'\
	| sort
