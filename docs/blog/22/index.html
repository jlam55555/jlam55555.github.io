A follow-up from the previous post...

Changes since then:

  • I have switched to using only left Shift, and it feels pretty natural now. As I thought when initially switching, it is much better for typing words or phrases in all caps.
  • I haven't gotten the hang of right thumb. I still always use left thumb.
  • I always use left pinky for Ctrl, Shift, and CapsLck, which is a lot of characters. As such, I've been trying to use left ring finger for CapsLck and Tab (I've never used pinky for tab anyways), or at least some combination of ring and pinky finger for CapsLck. (Maybe first keypress with pinky and successive ones with ring finger?)
  • My muscle memory is still pretty bad. I have to think about a lot of the keys before I type them, especially as I'm reading text.
  • I've heard opinions from other people in the Discord that ergonomic keyboards are no good at all for type racing. Standard keyboards are just as good. I think I agree, but the ergonomic layout still feels nice (that is what it's supposed to accomplish anyways).
